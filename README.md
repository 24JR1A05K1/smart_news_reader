# 📰 Smart News Reader

##  Overview
Smart News reader is an AI-powered web application that fetches the latest news articles using NewsAPI, organizes them into categories, and generates concise summaries using Natural Language Processing (NLP). The application provides users with a clean and responsive interface to browse news efficiently.

---

##  Objectives

- Fetch real-time news articles.
- Organize news into categories.
- Allow keyword-based news search.
- Generate AI-powered summaries.
- Display news in a responsive web interface.

---

##  Features

- Live news updates using NewsAPI
- Category-wise news (Technology, Business, Sports, Health, Entertainment)
- Keyword-based search
- AI-powered text summarization
- Responsive user interface
- Read Full Article option

## 🚀 How to Run

1. Install Python
2. Install required libraries

```bash
pip install -r requirements.txt
```

3. Add your NewsAPI key in `config.py`

```python
API_KEY = "YOUR_API_KEY"
```

4. Run

```bash
python app.py
```

5. Open

```
http://127.0.0.1:5000
```
